Quest-ce que SolutionVR ??==================

SolutionVR est un plugin Unity basé sur le plugin MiddleVR.
Utilisez votre application Unity dans n'importe quel système multi-écrans, scène virtuelle pour la production virtuelle ou systèmes VR tels que les murs, les grottes, etc.

À la base, MiddleVR est une bibliothèque gérant tous les aspects du multi-affichage
et le rendu en cluster, mais ajoute également en option des fonctionnalités de réalité virtuelle telles que :
périphériques d'entrée, stéréoscopie, interactions.

Plus d'infos sur MiddleVR: https://www.middlevr.com

Architectures matérielles prises en charge
--------------------------------

Cette version de SolutionVR prend en charges les projets sur unity 2018.3 sous MiddleVR 1.7
